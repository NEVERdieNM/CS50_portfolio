Ransomware



This was the first time I wrote a programm in Python except for when I first installed the Python intepreter and wrote a "Hello, World!" programm.
I got the inspiration for this project from a Youtube short which showcased how you can use the fernet module from the Python cryptography library to encrypt/decrypt a certain file.
I tried it myself, and then decided I want to bring it a step further and also use that as an opportunity to get more familiar with Python.
What I thought about adding was recursivity, so that is what I decided to add along with some features to make it more user friendly and "safe" in the sense that I implement some sort of failsafe or warning the user gets if for example the user tries to encrypt a file that has already been encrypted, since that would make it impossible to decrypt if the user didn't save the key of the previous encryption.

There are two programms, Encrypt.py and Decrypt.py.
Encrypt.py goes through all files and directories inside the directory in which the programm was executed and first checks if they have already been encrypted, if it finds a file that has already been encrypted it warns the user and asks if the program should continue execution.
It then lists all files that will be encrypted, if there are files that fail to encrypt, it also lists those files.
After the encryption it creates a key.txt file in which the key used to encrypt the files is stored.
Decrypt.py first reads the key.txt file, if it doesn't exist it notifies the user and stop execustion.
If key.txt was found it goes through all files and directories inside the directory in which the programm was executed and recursively decrypts all files.
It then prints all decrypted files, and all files that failed to decrypt or have been ecncrypted more than once (if there are any).
